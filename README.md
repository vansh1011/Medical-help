# Medical Help

A web app focused on helping users manage their medical data with the help of AI.

---

## Features

- Upload and store medical files and reports
- Files are saved and accessible anytime in the future
- AI integrated via API to assist users with their medical data

---

## Tech Stack

| Part | Technology |
|------|------------|
| Frontend | React, Vite, Axios |
| Backend | Node.js, Express |
| Database | MongoDB Atlas |
| Auth | JWT, bcrypt |
| AI | gemini API |
| Deployment | Vercel + Render |
